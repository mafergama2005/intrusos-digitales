# 🎮 Cazador de Intrusos Digitales

## 📌 Contexto
Videojuego desarrollado con Canvas API donde el jugador elimina intrusos digitales.

## 🎯 Objetivo
Eliminar la mayor cantidad de virus posibles.

## 💡 Justificación
Aplicar conceptos de programación gráfica, eventos y animación.

## ⚙️ Operación
- Click sobre enemigos
- Aumenta dificultad
- Guarda puntuación

## 🛠 Tecnologías
- HTML
- CSS
- JavaScript
- Bootstrap
- Canvas API
